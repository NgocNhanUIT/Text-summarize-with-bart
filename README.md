# Text Summarize with BART

## Installation:

Make sure you have Python and pip installed on your machine.

Install Flask and Transformers using pip:

```bash
pip install flask
pip install transformers
```

## Run the Web App:

To run the web application, execute the following command in your terminal:

```bash
python app.py
```

Visit the application in your web browser at http://127.0.0.1:5000/

Enjoy text summarization with BART!
